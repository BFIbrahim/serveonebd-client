## ‚ú® Key Features
- Secure Firebase authentication with role-based access control
- Separate dashboards for Admin, Volunteer, and User
- Users can request help, donate, and apply to become a volunteer
- Volunteers can create and manage donation campaigns
- Admin can manage pending requests, campaigns, urgent campaigns, and volunteers
- Fully responsive design for mobile and desktop devices

Although the frontend implementation was smooth due to prior experience, the backend introduced meaningful challenges that helped strengthen my skills.

Challenge 1: Designing Complex APIs  
As a full-stack project, creating and managing complex APIs required careful planning. By studying official documentation, exploring community forums, and testing different approaches, I improved my ability to design scalable and maintainable APIs.

Implementing JWT-based authentication required a solid understanding of token generation and verification. Reading the official JWT documentation helped me understand best practices and reinforced the importance of relying on official resources for deep technical learning.
